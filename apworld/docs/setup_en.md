# Warcraft 3 Archipelago Setup Guide
There are two components to the Warcraft 3 Archipelago randomizer, the apworld client and the map files.

## APworld setup
Get the .apworld file, and place it in your Archipelago installation in the custom_worlds/ folder.

You should then be able to generate a multiworld containing Warcraft 3 yamls, and launch the client.

## Mod setup
Download the map files from the release page on github.
Place them in the following folder: `<user>/Documents/Warcraft III/Maps/Archipelago`.

The maps should now appear under the Archipelago/ folder in the "Custom Game" menu within Warcraft 3.
The subfolder is necessary to have the next mission automatically start after beating the previous one.

## Playing the game
Simply open Warcraft 3 and launch any Archipelago map as a custom game
while the Archipelago client is running.
If the Archipelago client is not running, the game will display an error message
that it could not establish a connection.
The game will still be playable, but your progress will not be saved, locations will not be collected,
and unlocks will not work.
