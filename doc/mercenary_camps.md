### Class changes
* Mercenary camps have 12 button slots for mercs.
* Preplaced mercenary camps will also sell an item for a location. That takes 2 slots for 10 slots
  * (1 item + 1 change hero target)

Let's aim for 4~5 units, Dark Trolls can be an outlier

* Forest Trolls: add Warlord (5)
* Kobolds: add Kobold Tunneler? Add Rock Golem? ()
* Bandits + Wizards: add Dark Wizard; add Enforcer
* Nerubians + Frost Revenant: add Ice Revenant, add Nerubian Seer?
* Sea Critters: add Stormreaver Apprentice, Stormreaver Necrolyte
* Fel Eredar: add Eredar Sorcerer, Eredar Warlock
